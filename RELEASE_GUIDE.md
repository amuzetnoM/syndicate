## Creating the Release

### Step 1: Merge PR to Main

1. Review and approve the PR on GitHub
2. Merge `copilot/develop-new-ui-design` into `main`
3. Wait for CI to pass on main branch

### Step 2: Create Git Tag

```bash
# Checkout main branch
git checkout main
git pull origin main

# Create annotated tag
git tag -a v3.7.0 -m "Release v3.7.0 - Modern Web UI"

# Push tag to trigger workflows
git push origin v3.7.0
```

### Step 3: Create GitHub Release

1. Go to: https://github.com/amuzetnoM/gold_standard/releases/new

2. **Tag:** Select `v3.7.0` from dropdown (or create new tag)

3. **Release Title:** 
   ```
   Gold Standard v3.7.0 - Modern Web UI
   ```

4. **Description:** Copy content from `RELEASE_NOTES_v3.7.0.md`

5. **Pre-release:** Unchecked (this is a stable release)

6. **Set as latest release:** Checked

7. Click **Publish release**

## Installation for Users

### From Release

```bash
# Download and extract
wget https://github.com/amuzetnoM/gold_standard/archive/refs/tags/v3.7.0.tar.gz
tar -xzf v3.7.0.tar.gz
cd gold_standard-3.7.0

# Install
bash setup.sh

# Optional: Install web UI
bash web_ui/install.sh
```

### From Docker

```bash
# Pull image
docker pull ghcr.io/amuzetnom/gold_standard:v3.7.0

# Run with web UI
docker run -d \
  -p 5000:5000 \
  -v $(pwd)/data:/app/data \
  -v $(pwd)/output:/app/output \
  -e GEMINI_API_KEY=your_key \
  ghcr.io/amuzetnom/gold_standard:v3.7.0
```

### From PyPI (if published)

```bash
pip install gost==3.7.0
```

## Rollback Procedure

If issues are discovered after release:

### Delete Tag and Release

```bash
# Delete local tag
git tag -d v3.7.0

# Delete remote tag
git push origin :refs/tags/v3.7.0

# Delete release on GitHub
# Go to releases page and click "Delete" on v3.7.0
```

### Create Hotfix

```bash
# Create hotfix branch
git checkout -b hotfix/v3.7.1 v3.7.0

# Make fixes
# ... commit changes ...

# Create new release
git tag -a v3.7.1 -m "Release v3.7.1 - Hotfix"
git push origin v3.7.1
```
