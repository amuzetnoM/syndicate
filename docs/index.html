Quick Start

Up and Running in Minutes

Get started with automated setup scripts for Windows, macOS, and Linux.

1

Clone the Repository

Clone Gold Standard from GitHub to your local machine.

2

Run Setup Script

Execute the automated setup script for your platform. It handles Python, venv, and dependencies.

3

Add API Key

Edit .env and add your GEMINI_API_KEY from Google AI Studio (free tier available).

4

Launch

Run the CLI or GUI and start receiving AI-powered gold analysis.

Terminal 
# Clone repository
git clone https://github.com/amuzetnoM/gold_standard.git
cd gold_standard

# Windows PowerShell
.\scripts\setup.ps1

# Unix/macOS/Linux
chmod +x scripts/setup.sh && ./scripts/setup.sh

# Quick test (no AI required)
python run.py --mode daily --no-ai

# Full run with AI
python run.py

# Or launch GUI
python gui.py
