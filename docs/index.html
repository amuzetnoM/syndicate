Quick Start

Up and Running in Minutes

Get started with automated setup scripts for Windows, macOS, and Linux.

1

Clone the Repository

Clone Gold Standard from GitHub to your local machine.

2

Run Setup Script

Execute the automated setup script for your platform. Handles Python, venv, and all dependencies.

3

Configure LLM Provider

Add Gemini API key, or configure Ollama/llama.cpp for local operation. Set PREFER_LOCAL_LLM=1 for offline mode.

4

Launch

Run the CLI daemon or GUI and start receiving AI-powered gold analysis.

Terminal 
# Clone repository
git clone https://github.com/amuzetnoM/gold_standard.git
cd gold_standard

# Windows PowerShell
.\scripts\setup.ps1

# Unix/macOS/Linux
chmod +x scripts/setup.sh && ./scripts/setup.sh

# Configure LLM (choose one)
GEMINI_API_KEY=your-key              # Cloud provider
LLM_PROVIDER=ollama                   # Local Ollama server
LLM_PROVIDER=local                    # Direct llama.cpp
PREFER_LOCAL_LLM=1                    # Local-first mode

# Run
python run.py                         # Autonomous daemon
python run.py --no-ai                 # Test without AI
python gui.py                         # Launch GUI
