Usage

Command Reference

All operational modes from autonomous daemon to single-run analysis.

▶ Daemon Mode

  • python run.py Autonomous daemon (1 min interval)
  • python run.py --interval-min 5 Custom interval (5 minutes)
  • python run.py --interval 2 Legacy hours-based interval

■ Single Run

  • python run.py --once Run once and exit
  • python run.py --run Run all analysis once
  • python run.py --daily Quick daily journal only

☰ Interactive

  • python run.py --interactive Menu-driven mode selection
  • python gui.py Launch GUI dashboard
  • python run.py --status Show system status

⚙ Options

  • python run.py --no-ai Run without LLM integration
  • python run.py --lifecycle list List documents by status
  • python run.py --help Show all options
