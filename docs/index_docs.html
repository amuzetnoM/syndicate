Intelligent Scheduling System

Gold Standard v3.2 introduces an industry-leading intelligent task scheduling system that transforms the daemon from a simple periodic runner into a sophisticated autonomous execution engine.

Three Core Principles: (1) Immediate by Default ‚Äî tasks without dates execute immediately. (2) Context-Aware ‚Äî natural language date extraction. (3) Self-Healing ‚Äî automatic recovery from crashes and quota exhaustion.

Natural Language Date Extraction

The system automatically parses temporal references from task descriptions using advanced pattern matching:

Execution State Machine

Tasks follow a deterministic state machine with atomic transitions to prevent race conditions and ensure reliable execution:

CREATED

  • Task extracted
  • Date parsed

PENDING

  • Awaiting execution
  • Or scheduled time

IN_PROGRESS

  • Atomically claimed
  • Worker assigned

COMPLETED

  • Success
  • Result stored

FAILED

  • Max retries
  • Error logged
Atomic Task Claiming
# Prevents duplicate execution across processes
if db.claim_action(action_id, worker_id='daemon_1'):
    try:
        result = execute_task(action)
        db.update_action_status(action_id, 'completed', result)
    except Exception as e:
        # Release back to pending for retry
        db.release_action(action_id, reason=str(e))
else:
    print("Task already claimed by another process")

Retry & Recovery System

The system automatically handles API quota exhaustion with exponential backoff and recovers from crashes:

Exponential Backoff

30s ‚Üí 60s ‚Üí 120s ‚Üí 240s ‚Üí 480s ‚Üí 600s max. Up to 10 retries before failure.

Crash Recovery

On startup, stuck in_progress tasks are reset to pending. Never loses work.

Health Monitoring

Real-time metrics: ready tasks, scheduled future, stuck count, 24h success rate.

Atomic Operations

claim_action(), release_action(), update_status() ‚Äî all race-condition-free.

Quick Start

Terminal
# Clone and setup
git clone https://github.com/amuzetnoM/gold_standard.git
cd gold_standard

# Windows
.\scripts\setup.ps1

# macOS/Linux
chmod +x scripts/setup.sh && ./scripts/setup.sh

# Add API key to .env
GEMINI_API_KEY=your_key_here

# Run
python run.py --interval-min 5   # Daemon mode
python gui.py                    # GUI dashboard
