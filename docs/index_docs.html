Overview

Syndicate is a Precious Metals Intelligence System that combines market data ingestion, technical analysis, regime/strategy synthesis, memory/persistence, and optional AI-enhanced insights.

What's New (v3.5.2)

  • GSR reliability — Gold/Silver Ratio now uses a DB fallback to compute reliably when live prices are missing or invalid.
  • Console UX — Compact, color-coded module tags and reduced logger-name noise for clearer terminal output.
  • Tests & Docker — Unit tests added and Docker images published as ghcr.io/amuzetnom/syndicate:3.5.2.

Key Capabilities

  • Autonomous Operation — Daemon mode with configurable intervals (default: 4 hours)
  • Multi-Asset Analysis — Gold, Silver, DXY, 10Y Yields, VIX, S&P 500
  • Technical Indicators — RSI, ADX, Bollinger Bands, SMA/EMA via pandas_ta
  • AI Integration — Gemini, Ollama, llama.cpp with automatic fallback
  • Persistent Memory — Cortex system with prediction grading and trade simulation
  • Report Generation — Daily journals, weekly rundowns, monthly/yearly reports

Installation

Requirements

  • Python 3.10 - 3.14
  • SQLite (bundled with Python)
  • Optional: Ollama server for local LLM
  • Optional: Docker for containerized deployment

Clone and Setup

git clone https://github.com/amuzetnoM/syndicate.git
cd syndicate
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt

Environment Configuration

# .env file
GEMINI_API_KEY=your_api_key_here
NOTION_API_KEY=your_notion_key  # Optional
NOTION_DATABASE_ID=your_db_id   # Optional
LLM_PROVIDER=gemini             # gemini, ollama, or local

Quick Start

First Run
The system will automatically initialize the Cortex memory and database on first execution.

Start Autonomous Daemon

python run.py

Runs continuous analysis according to `RUN_INTERVAL_HOURS` (default: 4 hours). Press Ctrl+C to stop gracefully.

Single Analysis Run

python run.py --once

Launch GUI

python gui.py

Architecture

Syndicate follows a three-module design with clear separation of concerns:

Module Class Responsibility
Memory Cortex Persistent memory, prediction grading, trade simulation
Data QuantEngine Market data fetchers, TA indicators, chart generation
Strategy Strategist Regime detection, bias synthesis, AI augmentation

Data Flow

[yfinance] → QuantEngine → Strategist → Cortex (memory)
     |              ↑
     ↓              |
 reports/charts     |
     ↓              |
 db_manager ← run/gui → user
     ↑
     +—— scripts/ → [Notion DB]

LLM Providers

The FallbackLLMProvider pattern ensures resilient AI-powered analysis:

Mode Environment Variable Provider Order
Default (none) Gemini → Ollama → llama.cpp
Local-First PREFER_LOCAL_LLM=1 llama.cpp → Ollama → Gemini
Local Only LLM_PROVIDER=local llama.cpp only
Ollama Only LLM_PROVIDER=ollama Ollama only
Gemini Only LLM_PROVIDER=gemini Gemini only

Command Reference

Daemon Mode

Command Description
python run.py Autonomous daemon (1 min interval)
python run.py --interval-min 5 Custom interval (5 minutes)
python run.py --interval 2 Legacy hours-based interval

Single Run

Command Description
python run.py --once Run once and exit
python run.py --run Run all analysis once
python run.py --daily Quick daily journal only

Options

Flag Description
--interactive Menu-driven mode selection
--status Show system status
--no-ai Run without LLM integration
--lifecycle list List documents by status
--help Show all options
