Architecture (Overview)

This page documents system boundaries, the ingest pipeline, the LLM worker architecture, and observability integrations.

Ingest

Design: adapters · normalization · manifest · snapshot storage. Initial connectors: FRED, Rapid, MarketFlow, TradingEconomics, yfinance.

Processing

Workers: task queue · executor daemon · sanitizer/audit trail · publish pipeline.

Observability

Prometheus metrics · Grafana dashboard · Alertmanager rules for queue health and sanitizer flags.
